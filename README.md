## Ads Restful Apis

Simple Ads management API that shows ads and related tags/categories. 

## Installation
* `cp .env.example .env`
* `php artisan migrate`
* `php artisan db:seed`
* `php artisan:serve`

## Design Pattern Used

1- `service` pattern I added all business logic in separate classes each class contains only one responsibility.
